## Commands

-   Help

```
kyro --help
```

-   List available Bibles

```sh
kyro --list-bibles
```

-   Read/search a passage of scripture

```sh
kyro read John 3:16

# start from the beginning of the book
kyro read John

# search for a range of verses
kyro search John 3:16-18
```

-   Verse of the day

```
kyro today
```
